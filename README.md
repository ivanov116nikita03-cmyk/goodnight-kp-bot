# Good Night Show — Telegram Бот для КП (v2)

## Что умеет бот
- Спрашивает имя клиента
- Предлагает выбрать локацию: Большая студия, Малая студия, Выезд
- Для выезда запрашивает адрес
- Выбор блоков программы кнопками с настройкой длительности каждого
- Показывает итоговую сводку для проверки
- Генерирует и отправляет готовый .pptx файл

## Файлы которые нужно подготовить

Положи рядом с bot.py 4 файла шаблонов:

| Файл               | Что это                          |
|--------------------|----------------------------------|
| template_big.pptx  | Шаблон кп1 — большая студия      |
| template_small.pptx| Шаблон кп2 — малая студия        |
| template_vyezd.pptx| Шаблон кп3 — выезд               |
| slide3_ref.pptx    | Эталон слайда 3 (КП_Даша_18.06)  |

## Установка на Railway (бесплатно)

### Шаг 1 — Получи токен
Открой Telegram, найди @BotFather, напиши /newbot.
Придумай имя и username. Скопируй токен вида: 7123456789:AAFxxx...

### Шаг 2 — Зарегистрируйся на GitHub
Открой github.com, Sign Up, создай приватный репозиторий goodnight-kp-bot.

### Шаг 3 — Загрузи файлы
Add file — Upload files. Загрузи:
- bot.py
- requirements.txt
- Procfile
- README.md
- template_big.pptx
- template_small.pptx
- template_vyezd.pptx
- slide3_ref.pptx

Нажми Commit changes.

### Шаг 4 — Запусти на Railway
Открой railway.app, войди через GitHub.
New Project — Deploy from GitHub repo — выбери goodnight-kp-bot.
Вкладка Variables — Add Variable:
  Name: BOT_TOKEN
  Value: твой токен

Готово! Railway запустит бота автоматически.

## Проверка
Найди бота в Telegram, напиши /кп — бот начнёт задавать вопросы.

## Команды
- /кп или /kp — создать КП
- /cancel — отменить

## Логика слайда 3
Всегда берётся из slide3_ref.pptx (эталон КП_Даша_18.06).
Подставляются: дата, время, программа, стоимость, адрес.
Студия: адрес всегда Денисовский переулок 30, стр. 1, стоимость руб/чел.
Выезд: адрес клиента, стоимость общая (не с человека).

## Название файла
Формат: КП_Имя_дата_Локация.pptx
Пример: КП_Даша_18.06_Большая студия.pptx
